Build

  • 1Full control over the system: rename, select, translate as you wish 
    rotateBy(ligand, uniformRandomRotation(), ligcenter);
moveBy(ligand,[D 0 0]-ligcenter);
ligand.set('all','resname','MOL');
  • 2Quick system building: from PDB to solvated ionized system in few steps. 
    solvate(all,'minmax', [-30 -30 -30; 30 30 30]);
  • 3Membrane systems supported
    lipidz_up = mean(prot.coords(prot.atomselect('resname DUM and name O'),3));
lipidz_down = mean(prot.coords(prot.atomselect('resname DUM and name N'),3));
lipidz_center = 0.5*(lipidz_up+lipidz_down);
  • 4Dock your favorite ligand with a simple command using AutoDock VINA algorithm
    [ poses score ] = dock(protein,ligand)
  • 5CHARMM and AMBER forcefields covered
    charmmBuild(all, 'topo',topos, 'outDir','./build','execute',1,'saltConc','0,15');
amberBuild(all, 'topo',topos, 'outDir','./build','execute',1,'saltConc','0,15');

Run

  • 6Equilibration and production simulation inputs generation
    in = acemdRead('equilibration.m');
  • 7Submit simulations locally or over the Cloud
    s = Molecule('structure.pdb');
s.view

Analyze

  • 8Ensemble simulation analysis using Markov State Models (MSM)
    model = Model(data);
  • 9All sorts of analysis metrics available: contacts, distances, dihedral angles, secondary structure
    data = MetricDistance.project(simulations,'protein and name CA', ...
          'resname MOL and noh','metric','contacts');
  • 10Integration with VMD for quick visualization
    s = Molecule('structure.pdb');
s.view
  • 11Analysis compatible with GROMACS, ACEMD and NAMD
    fsims = simFilter(sims,'./filtered/','filter','not water');
