# System building

Build solvated, parameterized systems ready for production MD. Each tutorial is a working end-to-end build for a different class of system, in order of increasing complexity:

```{toctree}
:maxdepth: 1

01-protein
02-protein-ligand
03-openff-ligand
04-stapled-peptide
05-cyclic-peptide
06-bicyclic-peptide
07-membrane
08-protein-rna
```

The {doc}`system-building overview <../../explanation/system-building>` lays out the conceptual pipeline these tutorials walk through; the [API reference](../../reference/index.md) for {py:mod}`htmd.builder` and {py:mod}`htmd.builder.nonstandard` has the full argument lists.
